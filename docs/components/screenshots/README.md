# Component Screenshots

This folder contains screenshots of each component for documentation.

## Capturing Screenshots

To capture screenshots for documentation:

1. Run the showcase example:
   ```bash
   cargo run --example showcase
   ```

2. Navigate to the component tab you want to capture

3. Use your system's screenshot tool:
   - **Windows**: `Win + Shift + S` (Snipping Tool)
   - **macOS**: `Cmd + Shift + 4`
   - **Linux**: `gnome-screenshot -a` or similar

4. Save screenshots with the component name:
   - `button.png`
   - `card.png`
   - `checkbox.png`
   - etc.
